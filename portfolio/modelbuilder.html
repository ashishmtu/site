ArcGIS Pro Model Builder

Purpose: Learn Model Builder iterators
Semester: 2nd (Spring)
Project date: 06 Feburary 2024
Skills: ArcGIS, Model Builder, Mapping

Objective:
The objective of this exercise is to utilize ArcGIS ModelBuilder with CDC Wonder Data to create yearly hot spot maps of heart disease in the Midwest.

Methodology:
The methodology involves downloading heart disease data, importing it into ArcGIS, creating model iterators to process and analyze the data yearly, performing hot spot analysis, and animating the yearly hot spot feature class maps.

Results:
The exercise results in 22 yearly feature classes of heart disease data and corresponding hotspot analysis map, which are then animated to visualize changes over time.
